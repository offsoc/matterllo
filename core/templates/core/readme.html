Instruction to integrate Matterllo with Mattermost

Step 1: Authorization with Trello
  • Login with default account: admin/admin
  • Click the button Log in with Trello to connect to your Trello account
  • Click Allow button to authorize
  • Now you can visit to your board list by click Go to board list button
Step 2: Select a board that you want to receive notifications on Mattermost
Step 3: Create webhook on Mattermost
  • Visit your Mattermost Integration page, Select Incoming webhook
  • Click Add Incoming Webhook button on the top right conner
  • Enter necessary information then click Save
  • After finishing creating webhook on Mattermost, you will get a webhook url like below:
    •         https://YOUR_MATTERMOST_DOMAIN/hooks/WEBHOOK_ID
  • Copy this url and go to next step
Step 4: Add webhook to Matterllo
  • Turn back to Step 2, when you select a board, you will see this page
  • Paste Mattermost webhook url that you created in Step 3 and give it a short description, then click Next button
  • Then, you select list of events on Trello that Matterllo will send Notifications to Mattermost when the event happens
  • Finally, click Save button to complete
Step 5: Make a change in your Trello board and check Mattermost to see result
